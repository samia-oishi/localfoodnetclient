# Local Food Lovers Network

A community-driven MERN platform that connects food enthusiasts who love exploring local restaurants, street food, and home-cooked meals. Users can share their food experiences, post reviews with photos, and discover what others are enjoying nearby.

## Tech Stack

- **Frontend:** React 19, Vite, React Router v7, Tailwind CSS v4, DaisyUI, Swiper, React Toastify, React Icons
- **Auth:** Firebase Authentication
- **Backend:** Node.js, Express 5, MongoDB Atlas (separate repo)
